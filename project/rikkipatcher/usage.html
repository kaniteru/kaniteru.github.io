How to Apply the Patch Data to Game

1

Click the 'Apply patch to game' button in the main interface.

2

Click the 'Select Folder' button and choose the patch data folder from your computer.

Where can I download the patch data?

3

Review the selected patch information and click the 'Start Patch' button to begin the patching process.

How to Create Custom Patch Data

1

From the main menu, click on 'Extract data from game' to begin the extraction process.

2

When the extraction is complete, navigate to the 'extracted' folder that was created.

Copy this folder to a different location - this will be the base for your custom patch data.

You can rename the folder to something meaningful for your patch.

3

Open the files in your custom patch folder and modify them according to your preferences.

You can change dialogue options, UI elements, game settings, and more.

For examples and inspiration, check out this repository.

Important: Do not modify any files in the 'migration' folder! These are essential for the patching process to work correctly.
