How can I apply the patch data into game?

Step 1

Click the 'Apply patch to game' button.

Step 2

Click the 'Select Folder' button then, select patch data folder.

Step 3

Click the 'Start Patch' button.

How can I make the custom patch data?

Step 1

Use 'Extract data from game'.

Step 2

If extract finished, copy the 'extracted' folder then, paste into different place.

This folder is your custom patch data.

Renaming the folder if you want.

Step 3

Modify the files in the your custom patch data.

If you need an example, please refer to this repo.


Alert: Don't modify the files in 'migration' folder!
