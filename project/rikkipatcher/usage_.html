Step 1

Download patcher here or build

Step 2

If you installed game using steam, press the 'Find GameDirectory Automatically' button.


If not, press 'Select GameDirectory Manually' then, select the folder contains game executable(.exe) file.

Extract data from game

Extract all modifiable data from game.


The extracted data will be saved in 'extracted' folder where 'RikkyPatcher.exe' is located.

Apply patch to game

Modify game using imported custom patch data.


Press the 'Apply patch to game' button then, select the folder containing the modified data and press 'Start Patch' button.

How to make modify & apply custom patch data into game?

1. Use 'Extract data from game'


2. Copy 'extracted' folder


3. Paste copied folder into different place and renaming folder.


4. Modify the contents of the folders.
Currently, RikkiPatcher support 'dialogues' only.
(Do not change the filename or file extension!)


5. Click 'Apply patch to game' in RikkiPatcher.


6. Click 'Select Folder'.


7. Choose your modified custom patch data folder.


8. Click 'Start Patch'.
